# SmaliApiCallsToPostman

This is pretty basic converting Smali files containg api calls into postman format so you can easilly troubleshoot the calls. 

Please feel free to PR this if you can provide some further functionality and/or fix bugs.

## How to use

1) Git clone this repo
2) cd SmaliApiCallsToPostman
3) Save the Smali file as with a name `ApiCalls.Smali` in this directory
4) run `GetApiCalls.py` with python
5) open Postman and click import
6) either select the output.json file or copy and paste the contents into raw input
